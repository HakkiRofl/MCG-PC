# MCG-PC
***
### Инструкция по сборке
 ___Рекомендуется CMake 4.0 и С++20!___

Библиотеки _Glad_ и _GLFW_ подключены как подмодули, для работы используйте:

При клонировании проекта с подмодулями используйте:
    
    git clone --recurse-submodules <адрес_вашего_репозитория>

Если уже склонировали — инициализируйте подмодули:

    git submodule init
    git submodule update --recursive

Подмодули фиксируют конкретный коммит. Чтобы обновить библиотеку, перейдите в папку подмодуля и переключитесь на нужную ветку или коммит:

    cd third-party/glad
    git checkout main
    git pull
